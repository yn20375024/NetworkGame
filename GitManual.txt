ブラウザでリモートリポジトリを作成する
git開く、ちまちまとcdでc:からローカルリポジトリを作成したいところに飛ぶ
飛んだ先でinitコマンド実行後、リモートリポジトリのURLでcloneしローカルリポジトリを作成

こうすることでgithubのリモートリポジトリのデータを引っ張ることが可能

追加する際はまずローカルリポジトリで変更した後で、
ファイルをgit addし、その後git commitする
commit -m "メッセージ"またはエディタで上でメッセージ入力後escで抜けて:(コロン)を押してwqを押す
最後にgit push origin ブランチ名でpushできる
これでリモートリポジトリに変更を反映できる

ほかからアクセスするにはSettingsのManage accessでそのユーザを追加し認証する必要がある

git pullでローカルリポジトリを更新する事ができる

git update-git-for-windowsでアップデート
